# Setup Instructions

The automated installation of required tools failed. Please manually install the following:

## 1. Git
- **Download**: [https://git-scm.com/download/win](https://git-scm.com/download/win)
- **Install**: Run the installer with default options.

## 2. Node.js (LTS Version)
- **Download**: [https://nodejs.org/en/download/](https://nodejs.org/en/download/)
- **Install**: Run the installer with default options.

## 3. Python (Version 3.10 or higher)
- **Download**: [https://www.python.org/downloads/](https://www.python.org/downloads/)
- **Install**:
    - **IMPORTANT**: During installation, check the box **"Add Python to PATH"**.
    - Run the installer with default options.

## 4. Verification
After installing all three, close and reopen your terminal (or restart your computer) and run the following commands to verify:

```powershell
git --version
node --version
python --version
```

Once you have verified the installations, please let me know so we can proceed with setting up the project dependencies.
